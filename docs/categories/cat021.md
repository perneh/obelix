# Category 021 – ADS-B Target Reports

**Edition:** 2.7 (Eurocontrol Part 12)  
**Purpose:** Ground-station ASTERIX encoding of ADS-B Extended Squitter target reports.

## Overview

Category 021 carries **Automatic Dependent Surveillance – Broadcast (ADS-B)** information from a ground receiver to downstream surveillance systems. Typical content includes ICAO 24-bit target address, WGS-84 position, flight level, and callsign.

Obelix implements the **default Eurocontrol UAP** (FRN 1–49) with editable encoders for the most common operational items used in scenario simulation.

## Implemented data items

| Item | FRN | Obelix field | Notes |
|------|-----|--------------|-------|
| I021/010 Data Source Identification | 1 | `data_source` | Mandatory SAC/SIC |
| I021/040 Target Report Descriptor | 2 | `target_report_descriptor` | Primary + optional 1st extension (GBS, SIM, …) |
| I021/071 Time of Applicability for Position | 5 | `time_of_applicability_position` | Included when WGS-84 position is sent |
| I021/130 Position in WGS-84 | 6 | `wgs84` | When `position_resolution` = `low` |
| I021/131 High-Resolution Position | 7 | `wgs84` | When `position_resolution` = `high` (default) |
| I021/080 Target Address | 11 | `target_address` | 6-digit hex ICAO 24-bit address |
| I021/140 Geometric Height | 16 | `geometric_height_ft` | Optional (`include_geometric_height`) |
| I021/070 Mode 3/A Code | 19 | `mode3a` | Optional octal code (`include_mode3a`) |
| I021/145 Flight Level | 21 | `flight_level` | Optional (`include_flight_level`) |
| I021/077 Time of Report Transmission | 28 | `time_of_report_transmission` | Optional |
| I021/170 Target Identification | 29 | `target_identification` | Up to 8 IA5 characters |

All other UAP entries are listed in the UI panel for reference; use external tools for full Cat 021 records with velocity, quality indicators, trajectory intent, etc.

## Target address

Enter the **ICAO 24-bit address** as six hexadecimal digits, e.g. `4AC872`. The address type (ATP) in I021/040 should match:

- **ATP = 0** — standard ICAO aircraft address  
- **ATP = 3** — anonymous / non-ICAO (e.g. hostile or non-cooperative tracks)

## Position resolution

- **High (I021/131):** LSB = 180/2³⁰ degrees (~2 cm) — default for Baltic scenarios  
- **Low (I021/130):** LSB = 180/2²³ degrees (~2.4 m)

Either FRN 6 or FRN 7 is sent, not both.

## Wireshark

Set ASTERIX preferences to **Category 021 Edition 2.7** (or 2.6+) for best decode alignment. Implemented FRN order follows the Eurocontrol default UAP.
