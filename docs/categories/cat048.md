# Category 048 – Monoradar Target Reports

**Edition:** 1.32  
**Purpose:** Single-radar target reports (plots) with measured position and Mode 3/A.

## Overview

Category 048 is used for **monoradar target reports** — raw or lightly processed plots from one radar sensor. Each message typically describes one detection with polar range and azimuth relative to the radar, plus an optional SSR Mode 3/A code.

This is a common category for testing plot ingestion, radar simulators, and Wireshark ASTERIX dissectors.

## Implemented data items

All three items below are **always included** in every Obelix message:

| Item | FRN | Obelix field | Encoding |
|------|-----|--------------|----------|
| I048/010 Data Source Identifier | 1 | `data_source.sac`, `data_source.sic` | SAC + SIC |
| I048/040 Measured Position | 2 | `position.rho_nm`, `position.theta_deg` | RHO: 16-bit, LSB = 1/256 NM; THETA: 16-bit, LSB = 360°/2¹⁶ |
| I048/070 Mode 3/A Code | 3 | `mode3a` | 4-byte item; octal code 0–7777 converted to 12-bit BCD |

### Mode 3/A (octal)

Enter the code in **octal** notation (digits 0–7 only), e.g. `7000` or `1234`. Value `0` is encoded as no code / default padding.

### Measured position

- **Range (RHO):** nautical miles, 0–256 NM
- **Azimuth (THETA):** degrees, 0–360°

## Example output

Basic plot at 10 NM / 45°, Mode 3/A 7000 (SAC=1, SIC=1):

```text
30000FC001010A00B40070000000
```

Breakdown:

| Bytes | Meaning |
|-------|---------|
| `30` | Category 048 |
| `000F` | Length 15 bytes |
| `FC` | FSPEC — FRN 1, 2, 3 present |
| `0101` | SAC=1, SIC=1 |
| `0A00B400` | RHO=10 NM, THETA=45° |
| `70000000` | Mode 3/A item with code 7000 |

## Shared configuration

Load **Basic Plot** from the library (`configurations/cat048/basic-plot.json`).

## Wireshark

```text
asterix.category == 48 && udp.port == 8600
```

Expand **Measured Position in Slant Polar Coordinates** and **Mode 3/A Code in Octal Representation** in the packet tree.

## Limitations

Obelix does **not** encode plot characteristics, target address, calculated position, flight level, or other extended UAP items from the full Cat 048 specification. Only the three items listed above are supported.
