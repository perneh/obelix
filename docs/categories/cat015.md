# Category 015 – INCS Target Reports

**Edition:** 1.1 (Eurocontrol ASTERIX Part 28)  
**Purpose:** Independent Non-Cooperative Surveillance (INCS) target reports.

## Overview

Category 015 carries **INCS target reports** from passive/non-cooperative surveillance systems (e.g. multilateration using DVB-T/DAB/FM transmitters). Use together with **Category 016** to trace a target back to the contributing transmitter/receiver pair.

## Implemented data items

| Item | FRN | Obelix field |
|------|-----|--------------|
| I015/010 Data Source Identifier | 1 | `data_source` |
| I015/000 Message Type | 2 | `message_type`, `report_generation` |
| I015/015 Service Identification | 3 | `service_id` (optional) |
| I015/145 Time Of Applicability | 6 | `time_of_applicability` |
| I015/161 Track/Plot Number | 7 | `track_plot_number` |
| I015/400 Measurement Identifier | 12 | `measurement_id` (links to Cat 016) |
| I015/600 Horizontal Position (WGS-84) | 13 | `wgs84` when position format = WGS-84 |
| I015/625 Range | 20 | `range_azimuth.range_m` |
| I015/627 Azimuth | 22 | `range_azimuth.azimuth_deg` |

## Position format

| Value | Format |
|-------|--------|
| WGS-84 | Latitude/longitude (I015/600 P84 subset) |
| Range & azimuth | Sensor-centric range (m) and azimuth (°) |

## Limitations

Obelix implements a practical subset for testing. Not encoded: target descriptor, track status, Doppler, contour, velocity items, and extended covariance fields from the full Part 28 UAP.
