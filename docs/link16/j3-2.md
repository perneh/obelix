# J3.2 Air Track

Air track report for the tactical picture (NPG 2).

## Key fields

| Field | Description |
|-------|-------------|
| `source_ju` | Simulated C2 unit / JU sending the message |
| `track_number` | Track number (TN) assigned on the Link 16 network |
| `identity` | Friend / Hostile / Unknown / … |
| `platform` | Air platform type |
| `mode3a` | IFF Mode 3/A code |
| `position` | WGS-84 latitude and longitude |
| `altitude_ft` | Geodetic altitude in feet |
| `course_deg` / `speed_kts` | Kinematic data |

## Example

Send via API:

```bash
curl -X POST http://localhost:8000/api/link16/send/J3-2 \
  -H 'Content-Type: application/json' \
  -d '{
    "fields": {
      "source_ju": 10001,
      "npg": 2,
      "track_number": 101,
      "identity": 3,
      "platform": 1,
      "mode3a": 7777,
      "position": {"latitude_deg": 59.33, "longitude_deg": 18.05},
      "altitude_ft": 35000,
      "course_deg": 90,
      "speed_kts": 450
    },
    "host": "host.docker.internal",
    "port": 8700,
    "protocol": "udp"
  }'
```

Use the **Link 16** tab in Obelix for interactive editing.
