    ASTERIX · Scenario Builder

    Send one or more ASTERIX messages in sequence — for example a moving aircraft track or a series of radar plots.

    1. 1
      Create a message

      In ASTERIX → Message Editor, select a category (e.g. Cat 62) and configure the fields.

    2. 2
      Add steps to the scenario

      Each step is a snapshot of your message. Optionally enable Animate route so position changes over time.

    3. 3
      Configure transport & timing

      Set where to send (host/port) and how often the scenario repeats.

    4. 4
      Run

      Click Start Scenario to begin sending. Use Pause or Stop to control playback.

    Edit scenario as JSON file

    Scenarios are JSON files (same format as data/scenarios/ and scenarios/shared/). Export to edit in VS Code, vim, or any editor — then import back.

    Workflow: Download .json → edit externally → Import .json or paste below → Apply JSON → Run. Use Save locally & download to write to data/scenarios/ and get a copy on disk.

    Run scenario

    Sends each step in order. While a step has Animate route enabled, multiple messages are sent along the route at the configured interval.
