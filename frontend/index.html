    Scenario Builder

    A scenario sends one or more ASTERIX messages in sequence — for example a moving aircraft track or a series of radar plots.

    1. 1
      Create a message

      In Message Editor, select a category (e.g. Cat 62) and configure the fields.

    2. 2
      Add steps to the scenario

      Each step is a snapshot of your message. Optionally enable Animate route so position changes over time.

    3. 3
      Configure transport & timing

      Set where to send (host/port) and how often the scenario repeats.

    4. 4
      Run

      Click Start Scenario to begin sending. Use Pause or Stop to control playback.

    Steps
